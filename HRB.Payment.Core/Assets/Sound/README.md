# 音频文件说明

此目录用于存放语音播报所需的音频文件（MP3格式）。

## 必需的音频文件

### 数字音频（0-9）
- `zero.mp3` - 零
- `one.mp3` - 一
- `two.mp3` - 二
- `three.mp3` - 三
- `four.mp3` - 四
- `five.mp3` - 五
- `six.mp3` - 六
- `seven.mp3` - 七
- `eight.mp3` - 八
- `nine.mp3` - 九

### 单位音频
- `ten.mp3` - 十
- `hundred.mp3` - 百
- `thousand.mp3` - 千
- `ten_thousand.mp3` - 万
- `one_bilion.mp3` - 亿

### 其他音频
- `yuan.mp3` - 元
- `dot.mp3` - 点（小数点）

### 支付方式音频
- `zfb_pay.mp3` - 支付宝到账
- `wechat_pay.mp3` - 微信到账

### 提示音
- `pay.mp3` - 支付开始提示音
- `cancel_pay.mp3` - 支付取消提示音

## 注意事项

1. 所有音频文件必须是MP3格式
2. 音频文件需要设置为"Resource"类型（在项目文件中配置）
3. 如果缺少某个音频文件，系统会静默跳过，不会影响主流程
